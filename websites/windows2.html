Description

Microsoft Windows 2.0 was a direct sequel of Windows 1.0, it is most noticible change is the fact the the windows are no longer tiled, instead the windows could be stacked which caused a copyright lawsuit from Apple, but Microsoft won over the lawsuit. Also, it is the first Windows version to support Office (But this is a whole different topic). Apart from that nothing really changed. Windows 2.1 was a major update to Windows 2.0, hardware compatibility.
