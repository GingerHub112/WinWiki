Description

It was a major Windows update that revolutionized Windows (There is another one that revolutionized Windows further and one that was supposed to revolutionize Windows for Tablets but ended up being a nightmare for desktop users); It added further color support, it added some 32-bit support, MSDOS Executive in two different programs (The app launcher is now called Program Executive, the file manager is now Windows Explorer that we all know today, and MS-DOS Executive became deappreciated and discontinued).
